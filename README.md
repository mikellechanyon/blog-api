# Blog API - INF222

A robust RESTful Backend API designed for managing blog articles. This project was developed as part of the *INF222 (Backend Development)* course. It provides a complete interface for creating, reading, updating, and deleting articles using Node.js and SQLite.

## 🚀 Features

- *Full CRUD Operations*: Create, Read, Update, and Delete articles.
- *Search Functionality*: Deep search across titles and content.
- *Categorization*: Filter articles by categories or authors.
- *Interactive API Docs*: Built-in Swagger UI for testing endpoints.
- *Local Database*: Uses SQLite for lightweight, file-based data storage.

## 🛠️ Tech Stack

- *Runtime*: Node.js
- *Framework*: Express.js
- *Database*: SQLite3
- *Documentation*: Swagger (OpenAPI 3.0)
